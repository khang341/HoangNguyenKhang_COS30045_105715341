COS30045 Data Visualisation Exercises
SVG, D3 Charts, and Interactive Visualisations
Overview

This repository contains the work completed for COS30045 Data Visualisation.
The project includes three main exercises that demonstrate different techniques for creating data visualisations on the web.

The exercises focus on using SVG graphics and D3.js to present and explore data interactively.

The visualisations mainly use television energy consumption data to analyse patterns such as energy use, screen technology, and star ratings.

Exercise 4 – SVG and Basic D3 Charts

Exercise 4 introduces the basics of SVG graphics and simple D3 visualisations.

In this exercise:

SVG elements were used to draw shapes and graphics.

Basic charts were created using D3.

Data was loaded from CSV files and visualised on the webpage.

Examples created in this exercise include:

simple SVG drawings

basic bar charts

introductory D3 visualisations

This exercise helped build a foundation for creating more advanced charts in later tasks.

Exercise 5 – Multi Chart Webpage

Exercise 5 focuses on building a dashboard style webpage with multiple charts.

The goal was to create different types of visualisations using D3 and display them on the same page.

Charts included:

Scatter plot
Energy consumption vs star rating

Donut chart
Energy consumption by screen technology

Bar chart
Average energy consumption for 55 inch televisions by screen type

Line chart
Electricity spot prices over time

These charts demonstrate different ways to explore and communicate patterns in the data.

Exercise 6 – Interactive Visualisations

Exercise 6 adds interactivity to the charts.

Two interactive visualisations were developed:

Interactive Histogram

The histogram shows the distribution of television energy consumption.

Users can filter the chart by screen technology using buttons:

ALL

LED

LCD

OLED

The chart updates dynamically when a filter is selected.

Interactive Scatterplot

The scatterplot shows the relationship between:

star rating

energy consumption

Each point represents a television model.

Colours represent different screen technologies, and tooltips display the screen size when hovering over a point.

Technologies Used

The project was built using the following technologies:

HTML – webpage structure

CSS – styling and layout

JavaScript – interaction logic

D3.js – data visualisation library

SVG – scalable vector graphics for charts

Data

The datasets used in these exercises include television energy data and electricity price data provided as part of the course materials.

The data includes attributes such as:

screen technology

screen size

energy consumption

star rating

These datasets were used to demonstrate different visualisation techniques.

Learning Outcomes

Through these exercises, the following skills were developed:

creating SVG graphics

building charts with D3

loading and processing CSV data

designing dashboard style webpages

adding interactive elements such as filters and tooltips
