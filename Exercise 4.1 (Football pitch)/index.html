Exercise 4.1: Draw SVGs

This page shows a simple football pitch created with SVG. The design was inspired by a football field icon and then simplified into my own version with a cleaner layout and softer colours.

Simple Football Pitch

SVG Elements Used

Rect

Used for the main pitch, outer border, penalty boxes, goal boxes, and the goals.

Line

Used for the centre line that divides the field into two halves.

Circle

Used for the centre circle, centre spot, and both penalty spots.

Path

Used for the curved penalty arcs near both penalty areas.

Text

Used for the page title and the note under the football pitch.

SVG container

The full drawing is placed inside one SVG element with a viewBox so it scales nicely on different screen sizes.

Reference
