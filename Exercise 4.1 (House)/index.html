Exercise 4.1: Draw SVGs

This page shows a simple house and garden created using SVG shapes. The drawing includes several SVG elements such as rectangles, circles, lines, paths, and groups.

SVG House Drawing

SVG Elements Used

  • rect – house body, grass, sky, windows, door
  • circle – sun and door knob
  • ellipse – clouds and tree leaves
  • polygon – roof of the house
  • line – window dividers
  • path – garden path
  • text – title of the drawing
  • g – grouping windows and clouds

Reference

Some inspiration for SVG shapes and structure was taken from SVG examples and icon references.
